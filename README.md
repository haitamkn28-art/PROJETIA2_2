# Projet IA - Système CBIR (Content-Based Image Retrieval)

Projet de fin d'études réalisé à Collège Teccart. 
Ce système permet d'extraire des caractéristiques de texture (GLCM) et d'intensité pour classer des images d'animaux et effectuer une recherche par similarité (CBIR).

## Fonctionnalités
- Extraction automatique de caractéristiques.
- Classification via Random Forest.
- Interface utilisateur interactive avec Streamlit.
- Recherche par similarité (distances Euclidienne, Canberra, Cosinus).
